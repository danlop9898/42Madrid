DESCRIPCION
El proyecto trata de recrear la funcion printf. Para ello usaremos va_list y dependiendo
de que caracter sea el siguiente de % usaremos unas funciones o otras.
En todas contamos y retornamos la longitud igual que hace la funcion printf.

La funcion al llegar a % avanza una posicion donde depende del caracter que haya se dirige
a una funcion o otra.


INSTRUCCIONES
Se compila y se ejecuta como cualquier programa.
Hay un main aparte con pruebas de cada caso
Si es s , c o % ira a la función dedicada a escribir caracteres
Si es d o i irá a la función encargada de escribir numeros
Si es x , X o p irá a la función encargada del hexadecimal
Si es u ira a la funcion dodne lo pasamos al codigo ASCII.
